# Notes
A simple web application for managing notes (CRUD operations) using Node.js, Express, MongoDB, and EJS.

# Installation
```bash

# Install dependencies
npm install

# Start MongoDB
- mongod
- mongosh

# Run the application
node index.js
Visit http://localhost:5000
```
